html elements

This html document will demonstrate what you can do with some of the common html elements.

Setting up a document

These elements are used to set up a html document. You will not necessarily see them when the web page loads, but it is useful for sectioning and targetting with other languages such as CSS or JavaScript.

<!DOCTYPE html> is an open element which must be placed at the very beginning of a html document. It tells the browser that the document is html 5. It has no closing tag.

<html> is the root element, i.e. top level element. All other element must be nested between the html tags.</html>

<head>The head element contains the machine-readable information (metadata) about the document, for example, the document title, style sheets, etc.</head>

<body>The body tags contain/represent the contents of html document. There can only be one <body> element in a document.</body>

<title>The title element contains the document title which you want to have appear in the browser's title tab.</title>

<header>The header element contains introductory information about the document.</header>

<footer>The footer element contains footnotes, such as information about the author of the section, copyright data or links to related documents.</footer>

Displaying text

<p>This is the paragraph element.</p>

<span style="color:red"> The <span> element is used to wrap a small chunk of text inside a line, and can be used to group elements for styling purposes. For example, we have used <span> to turn the text in this section red.</span>

<div>The <div> element is used to divide blocks of text - it doesn't do much on its own (apart from gives a break between sections without the use of <br>), but becomes a useful container when combined with other codes.</div>

The <q> element indicates that the text is a quotation. In most browsers the text will be <q>enclosed in quotation marks.</q>

<code>The code element defines a fragment of computer code. The text surrounded by it is typically displayed in a monospace font.</code>

Images and Links (or anchors)

The <img> element lets you add an image to your webpage.
It has no closing tag, and the source (src) must be specified.
The alt attribute is what appears in the tooltip when you hover over the image. It is best practice to add this, for accessibility and if the image does not load correctly.
<img src="\home\arianha\Downloads\sushi.jpg" alt="a plate of sushi" height="200" width="300"/>

The anchor element allows you to add a hyperlink to your webpage.
It must list the address of the link using the href attribute.
The code
<a href=”https://www.manchestercodes.com/”>Manchester Codes</a>
will create a link to the Manchester Codes homepage.

Tables

You can create tables in html. Below code will give you a simple table, with three columns and five rows.
<table>
  <tr>
     <th><b>Name</b></th>
     <th><b>Favourite Colour</b></th>
     <th><b>Favourite drink</b></th>
  </tr>
  <tr>
     <td>Arianha</td>

  </tr>
  <tr>
     <td>Alex</td>

Forms & Buttons

Forms can be created using the form element.
E.g. <form action="/url-where-you-want-to-submit-form-data"></form>
Inside the form element should be an <input> element to create a section where the user can input their ideas.
You can also add a button to submit the form.
